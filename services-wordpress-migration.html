SEO Preservation

We take SEO seriously. Here's how:

  • 301 Redirects: Every old URL redirects to the new equivalent, preserving link equity
  • Metadata Transfer: Page titles, meta descriptions, and structured data are preserved
  • Canonical Tags: Properly configured to avoid duplicate content issues
  • Sitemap & Robots.txt: Updated to reflect the new site structure
  • Google Search Console: We help you update the property and request re-indexing

What Happens to Your Data

  • All content is backed up and transferred securely
  • Old WordPress database is archived (you keep a copy)
  • All images are optimised, converted to WebP, and hosted on the CDN
  • You receive the complete site code and can access it anytime

Common Questions

Will I lose my search rankings during migration?

No. We set up 301 redirects for every page, which preserves your existing rankings. Google recognises the redirects and transfers ranking authority to the new URLs. You may see a slight temporary dip, but rankings should recover within weeks.

Can I still edit my site after launch?

Yes. You receive the full source code and can edit it yourself with a text editor. Alternatively, we can teach you how to update content, or you can hire a developer familiar with static sites. For ongoing changes, we offer the $300/month support package.

What if I need a blog?

We build blogs into static sites using HTML and can integrate them with simple CMS solutions if you prefer dashboard editing. We'll recommend the best approach for your needs.

Will everything look different?

Not necessarily. We can preserve your existing design and just improve the performance underneath. Or, if you want a refresh, we can modernise the aesthetics as part of the migration. Your choice.

What if I change my mind after launch?

You own the code 100%. You can take it elsewhere anytime, migrate back to WordPress, or publish it yourself. No lock-in, no penalties.

Does hosting cost extra?

No. Hosting is included in your project price and for the first year after launch. After that, hosting via Vercel is optional but very affordable (typically $20–$50/month depending on traffic, or free for very low traffic sites).

How do I manage email and domains?

Your domain and email are separate from your website. You can keep your current domain registrar and email provider. We'll help with DNS setup if needed.

What if my WordPress site has custom plugins or complex functionality?

We assess each custom feature. Many plugins can be replaced with simpler solutions or built directly into the static site. If something isn't feasible, we'll let you know upfront and discuss alternatives.

Eligibility Checklist

WordPress migration is ideal if your site:

  • Has stable, well-organized content (pages, blog posts, images)
  • Doesn't require frequent user registrations or account management
  • Doesn't need real-time data or stock inventory management
  • Has reasonable file size (images aren't extremely large or numerous)
  • Uses standard WordPress structure (custom post types are fine but must be scoped)
