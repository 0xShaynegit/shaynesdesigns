# Local Fonts

This directory contains local font files for Shayne's Designs. Download the following fonts from Google Fonts and place the WOFF2 files here:

## Fonts to Download

### Syne (Display Font)
https://fonts.google.com/specimen/Syne

Download weights: 400, 600, 700, 800

Files needed:
- syne-400.woff2
- syne-600.woff2
- syne-700.woff2
- syne-800.woff2

### Cormorant Garamond (Editorial Font)
https://fonts.google.com/specimen/Cormorant+Garamond

Download weights: 300, 400, 600 (regular and italic)

Files needed:
- cormorant-garamond-300.woff2
- cormorant-garamond-300-italic.woff2
- cormorant-garamond-400.woff2
- cormorant-garamond-400-italic.woff2
- cormorant-garamond-600-italic.woff2

### DM Sans (Body Font)
https://fonts.google.com/specimen/DM+Sans

Download weights: 300, 400, 500

Files needed:
- dm-sans-300.woff2
- dm-sans-400.woff2
- dm-sans-500.woff2

## How to Download

1. Visit each Google Fonts link above
2. Select the required weights
3. Click Download
4. Extract the WOFF2 files
5. Place them in this directory with the names specified above

All font files are declared in `fonts.css` with `@font-face` rules.
